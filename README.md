# Defensive Prior Art

Patent-grade invention disclosures published to the public domain.

## Purpose

These documents describe inventions that **should exist but don't yet**. By publishing them here with verifiable timestamps (git commit SHAs), we establish that these ideas were publicly disclosed and **cannot be patented** by any party under [35 U.S.C. § 102(a)(1)](https://www.law.cornell.edu/uscode/text/35/102).

## What Makes This Prior Art?

For a publication to constitute prior art:
1. **Public disclosure** — this repository is public and indexed by search engines
2. **Verifiable date** — every git commit has a cryptographic SHA and timestamp
3. **Enabling disclosure** — each invention is described in sufficient technical detail that a person having ordinary skill in the art (PHOSITA) could implement it
4. **Real references** — every cited patent, paper, and data source is real and verifiable
